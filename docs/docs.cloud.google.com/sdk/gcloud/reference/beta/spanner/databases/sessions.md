NAME

gcloud beta spanner databases sessions - manage the sessions for Cloud Spanner databases

SYNOPSIS

`  gcloud beta spanner databases sessions  ` `  COMMAND  ` \[ `  GCLOUD_WIDE_FLAG â€¦  ` \]

DESCRIPTION

`  (BETA)  ` Manage the sessions for Cloud Spanner databases.

GCLOUD WIDE FLAGS

These flags are available to all commands: `  --help  ` .

Run `  $ gcloud help  ` for details.

COMMANDS

`  COMMAND  ` is one of the following:

  - `  delete  `  
    `  (BETA)  ` Delete a Cloud Spanner session.
  - `  list  `  
    `  (BETA)  ` List the Cloud Spanner sessions contained within the given database.

NOTES

This command is currently in beta and might change without notice. These variants are also available:

``` text
gcloud spanner databases sessions
```

``` text
gcloud alpha spanner databases sessions
```
