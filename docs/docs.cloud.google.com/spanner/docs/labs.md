**PostgreSQL interface note:** Some of the following codelabs aren't supported for PostgreSQL databases.

**Getting started**

  - [](https://codelabs.developers.google.com/codelabs/cloud-spanner-first-db)
    
    ### Spanner: Your first database open\_in\_new
    
    Learn how to set up a Spanner instance, and create your first database in that instance.

  - [](https://codelabs.developers.google.com/codelabs/spanner-graph-getting-started)
    
    ### Spanner: Get started with Spanner Graph open\_in\_new
    
    Learn how to set up a Spanner Graph database and query graph data with Graph Query Language (GQL).

  - [](https://codelabs.developers.google.com/codelabs/spanner-getting-started-vector-search)
    
    ### Spanner: Get started with vector search open\_in\_new
    
    Learn how to set up a Spanner database to perform vector search and access embedding and LLM models from Vertex AI's model garden using SQL.

  - [](https://codelabs.developers.google.com/codelabs/spanner-cassandra-adapter-getting-started)
    
    ### Spanner: Getting started with Spanner Cassandra Adapter open\_in\_new
    
    Learn how to set up a Spanner database and how to migrate your NoSQL database from Cassandra to Spanner.

**Use-case specific**

  - [](https://codelabs.developers.google.com/spanner-online-banking-app)
    
    ### Create an online banking application with Spanner open\_in\_new
    
    Using an online banking application, build upon your understanding of Spanner and dive into leveraging its advanced integrations to enhance your application's data processing and analytical capabilities.

  - [](https://codelabs.developers.google.com/spanner-ai-graph-recommendations)
    
    ### Real-time retail recommendations with Spanner AI and Graph open\_in\_new
    
    Use Spanner to transform your product catalog into a recommendation engine that can find products using natural language, explain why a product is a good fit, and discover related items.

  - [](https://codelabs.developers.google.com/patent-search-spanner-gemini)
    
    ### Spanner: Build a patent search application with Spanner, Vector Search and Gemini 1.0 Pro open\_in\_new
    
    Learn how to search for patents faster, in a more intuitive way, and with precision by leveraging Spanner, in-place Gemini 1.0 Pro, embeddings, and Vector Search.

  - [](https://codelabs.developers.google.com/codelabs/cloud-spanner-game-tradepost)
    
    ### Spanner: Game trading post open\_in\_new
    
    Learn how to create two Go services that interact with a regional Spanner database to enable players to acquire items and money, and then list items on the trading post for other players to purchase. This lab is dependent on completing the previous Spanner: Getting started with games development codelab.

  - [](https://codelabs.developers.google.com/codelabs/cloud-spanner-getting-started-game)
    
    ### Spanner: Getting started with games development open\_in\_new
    
    Learn how to create two Go services that interact with a regional Spanner database to allow players to sign up and start playing.
