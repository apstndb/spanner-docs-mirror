## Get a Google support package

Google Cloud offers different support packages to meet different needs, such as 24/7 coverage, phone support, and access to a technical support manager. For more information, see [Cloud Customer Care](/support) .

## Ask a question on Stack Overflow

Ask a question about Spanner on [Stack Overflow](http://stackoverflow.com/questions/tagged/google-cloud-spanner) . Use the tag `  google-cloud-spanner  ` for questions about Spanner. This tag not only receives responses from the Stack Overflow community, but also from Google engineers. We aim to answer all questions in a reasonable period of time.

## Discuss Spanner

Visit the Google Cloud [Slack community](https://googlecloud-community.slack.com/) to discuss Spanner and other Google Cloud products. If you haven't already joined, [use this form to sign up](https://join.slack.com/t/googlecloud-community/shared_invite/zt-3icuwfdvq-eq_58LqmOoCNc16mpZcaNA) . For Spanner, join the [`  #cloud-spanner  `](https://googlecloud-community.slack.com/messages/C49R7DSTH/) channel.

## File bugs or feature requests

File new feature requests and bug reports, or follow existing ones, using the [Spanner issue tracker](https://issuetracker.google.com/issues?q=componentid:190851%2B%20status:open) .
